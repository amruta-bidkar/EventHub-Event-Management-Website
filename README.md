# EventHub – Event Management Website 🎉

EventHub is a responsive and user-friendly event management website designed to help users explore, plan, and organize events efficiently. The project focuses on a clean UI, smooth navigation, and modern web design practices.

## 🔹 Features
- Responsive navigation bar
- Attractive hero section with call-to-action
- Event exploration page
- Clean and modern UI design
- Mobile-friendly layout
- Organized and reusable code structure

## 🔹 Technologies Used
- HTML5
- CSS3
- Responsive Web Design

## 🔹 Project Purpose
This project was created to practice frontend web development skills and to demonstrate the implementation of a real-world event management website layout.

## 🔹 Future Enhancements
- Login and Signup functionality
- Dynamic event listing using JavaScript
- Backend integration with Django / Flask
- Admin dashboard for managing events
- Contact form with validation
